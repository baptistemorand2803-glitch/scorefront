Bienvenue! Ce site présente les résultats des matchs de qualifications pour la coupe du monde de basketball 2027 du groupe G (France, Finlande, Belgique & Hongrie).

Les données affichées proviennent d'une base MySQL en ligne, accessible via une API Node.js déployée sur Render.
